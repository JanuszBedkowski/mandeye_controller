# SlavePPS

Synchronises system clock on a slave Raspberry Pi using a GPS NMEA stream and a PPS signal generated by the master Pi's **fake PPS** service.

## How it works

1. Reads NMEA sentences from a UART-connected GPS (or fake-GPS source).
2. Parses `$GPRMC` / `$GNRMC` sentences with **minmea** to extract UTC time.
3. Waits for the next PPS pulse via `/dev/ppsX`.
4. Sets `CLOCK_REALTIME` to the next whole GPS second, aligned to the PPS edge.

## Requirements

- `linuxpps-tools` / kernel PPS support (`pps-gpio` or `pps-ldisc` overlay enabled)
- `minmea` library
- Binary installed at `/opt/mandeye/extras/mandeye_slave_pps`
- Must run as **root** (required for `clock_settime`)

## Raspberry Pi setup

### 1. Enable UART and PPS overlay

Add to `/boot/firmware/config.txt` (adjust GPIO pin as needed):

```
enable_uart=1
dtoverlay=pps-gpio,gpiopin=18
```

Reboot.

### 2. Verify devices

```bash
ls /dev/ttyAMA0   # NMEA input
ls /dev/pps0      # PPS device
```

### 3. Install and start the service

```bash
sudo cp services/mandeye_slave_pps.service /usr/lib/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable mandeye_slave_pps
sudo systemctl start  mandeye_slave_pps
```

### 4. Check status

```bash
sudo systemctl status mandeye_slave_pps
sudo journalctl -fu mandeye_slave_pps
```

## Configuration

All settings are via environment variables. Override defaults by creating a drop-in:

```bash
sudo mkdir -p /etc/systemd/system/mandeye_slave_pps.service.d/
sudo tee /etc/systemd/system/mandeye_slave_pps.service.d/override.conf <<EOF
[Service]
Environment="SLAVE_PPS_UART_PORT=/dev/ttyS0"
Environment="SLAVE_PPS_UART_BAUD_RATE=115200"
Environment="SLAVE_PPS_DEVICE=/dev/pps1"
EOF
sudo systemctl daemon-reload && sudo systemctl restart mandeye_slave_pps
```

| Variable                  | Default         | Description              |
|---------------------------|-----------------|--------------------------|
| `SLAVE_PPS_UART_PORT`     | `/dev/ttyAMA0`  | UART device for NMEA     |
| `SLAVE_PPS_UART_BAUD_RATE`| `9600`          | Baud rate                |
| `SLAVE_PPS_DEVICE`        | `/dev/pps0`     | PPS character device     |
